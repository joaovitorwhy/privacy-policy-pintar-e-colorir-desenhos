Política de Privacidade

Pintar e Colorir Desenhos  ·  Última atualização: maio de 2026

Esta Política de Privacidade descreve como o aplicativo Pintar e Colorir Desenhos coleta, usa e protege as informações dos usuários.

1. Informações coletadas

O aplicativo não coleta dados pessoais identificáveis. As informações armazenadas ficam exclusivamente no dispositivo do usuário, incluindo:

  • Desenhos salvos pelo usuário (imagens armazenadas localmente)
  • Preferências de uso do aplicativo

2. Uso de dados

Os dados armazenados localmente são utilizados apenas para:

  • Exibir os desenhos salvos na galeria do aplicativo
  • Permitir que o usuário continue editando desenhos anteriores

3. Compartilhamento de dados

Não compartilhamos, vendemos ou transferimos dados do usuário para terceiros. Nenhuma informação é enviada para servidores externos.

4. Crianças

O aplicativo é voltado para o público infantil. Não coletamos intencionalmente informações pessoais de crianças. Como nenhum dado pessoal é coletado, o aplicativo está em conformidade com as diretrizes de proteção à privacidade infantil (COPPA e similares).

5. Permissões

O aplicativo pode solicitar as seguintes permissões:

  • Armazenamento: para salvar e carregar desenhos no dispositivo

6. Segurança

Como todos os dados permanecem no dispositivo do usuário e não há transmissão de informações, o risco de exposição de dados é mínimo.

7. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que os usuários revisitem esta página para verificar eventuais mudanças.
